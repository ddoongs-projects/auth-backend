== 개요

이 API문서는 'ddoongs-auth' 프로젝트의 산출물입니다.

=== API 서버 경로

[cols="2,5,3"]
|====
|환경         |DNS |비고
|개발(dev)    | link:[추후 확정 예정] |API 문서 제공
|운영(prod)   | link:[추후 확정 예정] |API 문서 미제공
|====

[NOTE]
====
해당 프로젝트 API문서는 개발환경까지 노출되는 것을 권장합니다. +
====

[CAUTION]
====
운영환경에 노출될 경우 보안 관련 문제가 발생할 수 있습니다.
====
